Crying Counter

The Crying Counter is a discord bot developed with Python. It keeps track of how many times a user says "I'm crying" and accounts for similar variations such as I am crying and typos by using the Levenstein Distance Algorithm. If the message is close to the original within a certain threshold then the user's count increments The counter for each user is tracked by using a diciontary. The discord username is the key and the value is the number of times they say a message that registers within being above a certain similiarity threshold then the user's count increments. An Sqlite3 file stores all the user information in the event that the bot crashes to ensure data isn't lost. User's can check their personal and other user's counter by using the !print_count @username command to check. Originally to account for more variations of the message, GPT API was integrated but this version of the bot wasn't feasible to implement as hundreds of messages are sent in the server everday and making a call to gpt quickly got expensive.
