Notable Projects

Crying Counter

The Crying Counter is a Discord Bot hosted in CSUF's ACM club.
The Crying Counter keeps track of how many times a discord user says "I'm crying" and similar phrases using the Levenstein Distance Algorithm to account for typos

To-Do Applicatoin

The ToDoApplication(Name is still a work in progress) is a one piece productivity themed app where users can organize their tasks for the day and update their calander.

University Database

The University Database project is a scaled down version of how a University Database. It keeps track of students, grades professors, courses, sections and can perform queries to pull up records of students and professors
