Notable Projects

Crying Counter

The Crying Counter is a Discord Bot hosted in CSUF's ACM club. It keeps track of how many times a Discord user says "I'm crying" and similar phrases using the Levenshtein Distance Algorithm to account for typos.

To-Do Application

The To-Do Application (Name is still a work in progress) is a One Piece productivity-themed app where users can organize their tasks for the day and update their calendar.

University Database

The University Database project is a scaled-down version of a university database. It keeps track of students, grades, professors, courses, and sections and can perform queries to pull up records of students and professors.
